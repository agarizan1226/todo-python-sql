# To-Do List

Ein einfaches Konsolenprogramm zur Verwaltung von Aufgaben.  
Die Aufgaben werden in einer SQLite-Datenbank gespeichert.

## Technologien

- Python
- SQL
- SQLite

## Funktionen

- Aufgaben anzeigen
- Aufgaben hinzufügen
- Aufgaben als erledigt markieren
- Aufgaben löschen

## Start

```bash
cd todo-python-sql
python3 main.py
